Get started

api

  1. Go to api directory and create a .env file based on the .env.example file.

frontend

  1. Go to frontend directory and create a .env file based on the .env.example file.

Docker compose

docker-compose up --build

you need to go inside your API container and do the following command to generate your prisma client to node_modules

npx prisma generate npx prisma db push

Restart your api container

  1. Navigate to the following URLs in your browser according to the project architecture :

    • http://localhost for the front

    • http://api.localhost for the GraphQL API (appolo server)

    • http://adminer.localhost for the database adminer
